在vue中集成markdown

2018-11-30
前端 Vue
约 476 字 预计阅读 1 分钟

文章目录

    • 我的失败经历
    • 使用

为了在自己的vue中集成markdown真的试遍了所有的方法,终于成功了,特此发布博文记录一下.

我的失败经历

  1. 使用vue-markdown + highlightjs 放弃原因只支持渲染markdown不支持编辑,并且试遍了网上的方法无法实现高亮,具体错误是无法解析"el”

  2. 使用vue-markdown-highlight + highlightjs 这个也是只支持渲染,按照官方的做法也无法实现高亮

  3. mavonEditor插件 这个插件可实现markdown的编辑和单独渲染,但是貌似有一些小bug.设置props的defaultOpen为preview,preview无法解析,这就意味着没法单独渲染markdown,所以放弃

  4. tui.editor 我的评价只能说,完美且强大 原版 vue版
